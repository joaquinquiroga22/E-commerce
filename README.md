# Henry


## Vivero E-commerce

This is a student project we made as part of our education at [Soy Henry](https://www.soyhenry.com/), which consisted of creating a full-stack React application from scratch.

### Our stack of technologies:

- Front End:
    + HTML - CSS - Javascript
    + React
    + React-Material-ui
    + Redux

- Back End:
    + Node.js
    + Express
    + Passport
    + Sequelize

- Database: 
    + PostgreSQL
